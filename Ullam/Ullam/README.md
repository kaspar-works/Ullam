# Ullam

A private journaling app for iOS and macOS.

## Overview

Ullam is a personal diary application designed for daily reflection and emotional tracking. The app supports multiple private diaries, rich text entries, and a unique calendar visualization that maps your emotional journey throughout the year.

## Features

### Rich Journal Entries
- Multiple entries per day
- Text formatting: title, subtitle, bold, italic, underline, and quotes
- Embed audio, images, and videos within entries

### Emotional Tracking
- Attach up to 3 emojis per entry to capture how you feel
- Assign a single emoji to summarize each day
- Yearly calendar grid displaying daily mood emojis at a glance

### Multiple Diaries
- Create named diaries (default diary: "Me & Me")
- Each diary can have its own pincode
- Pincode-protected diaries are fully encrypted
- Hidden diary count — no indication of how many diaries exist
- Switch between diaries by entering different pincodes
- App opens to the current day in your default diary

### Storage Options
- Local device storage
- iCloud sync (user preference)

## Platforms

- iOS
- macOS
